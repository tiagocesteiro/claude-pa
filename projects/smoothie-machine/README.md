# 7-Eleven Smoothie

Automated self-service frozen smoothie model targeting Lisbon gyms and co-working spaces.

**Status:** Active -- evaluating feasibility

## Description

- Pre-packed frozen cups blended in 60s using HM-160E tabletop machine
- Cost ~€1.20/cup | Retail €4.50-€5.50 | Margin ~70%+
- Partner model: 20-30% profit share with venue
- PLA compostable cups (compliant with 2026 single-use plastic ban)

## Target Locations

- Climbing gyms: Vertigo, Escala25
- Co-working: IDEA Spaces, Avila Spaces

## Key Dates

- 2026: Single-use plastic ban -- PLA cups required
- Target: feasibility decision by end of March 2026
