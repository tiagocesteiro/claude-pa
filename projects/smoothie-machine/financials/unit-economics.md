## Key Drivers
- **Venue footfall:** More members = more sales potential
- **Pricing power:** Gyms/co-works can command premium (€5+)
- **COGS control:** Supplier sourcing is critical — target <€1.10/cup
- **Venue share:** Negotiate lower than 25% if possible (20% is healthier)
