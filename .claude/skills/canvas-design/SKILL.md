# Canvas Design Skill

Create visual art through a structured two-step process combining design philosophy with visual expression.

## Core Approach

1. Develop a named aesthetic movement with a 4-6 paragraph philosophy
2. Express it as a museum-quality, single-page visual composition

## Key Principles

- **Visual Priority**: 90% visual design, 10% essential text
- **Craftsmanship**: Work must appear meticulously crafted, product of deep expertise
- **Conceptual Depth**: Embed subtle references that function as abstract composition for general audiences
- **Originality**: Never copy existing artists; create original work

## Output Formats

- `.md` — design philosophy document
- `.pdf` or `.png` — visual expression

## Execution Standards

- Pristine execution with painstaking attention to spacing, typography, alignment
- Minimal, visually-integrated text
- Treat design as fine art, not functional decoration
- Expert-level visual communication over explanatory content
