# Theme Factory Skill

Curated professional font and color themes to apply to any artifact (slides, docs, HTML pages, reports).

## Usage

1. Show `theme-showcase.pdf` to the user — display without modification
2. Ask which theme to apply
3. Wait for explicit selection
4. Apply selected theme's colors and fonts

## Available Themes

1. **Ocean Depths** — Professional maritime
2. **Sunset Boulevard** — Warm vibrant sunset
3. **Forest Canopy** — Natural earth tones
4. **Modern Minimalist** — Clean grayscale
5. **Golden Hour** — Rich autumnal palette
6. **Arctic Frost** — Cool winter-inspired
7. **Desert Rose** — Soft dusty tones
8. **Tech Innovation** — Bold modern tech
9. **Botanical Garden** — Fresh organic colors
10. **Midnight Galaxy** — Dramatic cosmic tones

Full specifications (hex codes, font pairings) in `themes/` directory.

## Application Process

1. Read the corresponding theme file from `themes/`
2. Apply colors and fonts consistently throughout the artifact
3. Ensure proper contrast and readability
4. Maintain visual identity across all elements

## Custom Themes

If no existing theme fits, generate a custom one based on user input. Give it a descriptive name, choose appropriate colors/fonts, show for review, then apply.
