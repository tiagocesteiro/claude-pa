# Anthropic Brand Styling Guidelines

Official brand identity standards for visual artifacts.

## Color Palette

- **Primary text**: `#141413` (dark)
- **Light background**: `#faf9f5` (cream)
- **Accent orange**: `#d97757`
- **Accent blue**: `#6a9bcc`
- **Accent green**: `#788c5d`

## Typography

- **Headings (24pt+)**: Poppins — fallback: Arial
- **Body text**: Lora — fallback: Georgia

No font installation required; system falls back gracefully.

## Application

Apply these colors and fonts consistently across slides, docs, HTML artifacts, reports, and any visual output. Use RGB values for technical implementation. Maintain visual hierarchy with proper contrast ratios.
