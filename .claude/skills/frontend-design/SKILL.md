# Frontend Design Skill

Create high-quality, distinctive web interfaces that avoid generic AI aesthetics.

## Design Philosophy

Choose a **bold aesthetic direction** before coding: brutalist, maximalist, minimalist, retro-futuristic, or another intentional style. Intentionality over intensity.

## Core Guidelines

- Choose fonts that are beautiful, unique, and interesting — avoid Arial and Inter
- Commit to cohesive color palettes with dominant colors and sharp accents
- Use animations strategically for high-impact moments, not scattered effects
- Create unexpected spatial compositions with asymmetry, overlap, generous spacing
- Add contextual visual details (textures, gradients, patterns) matching the aesthetic

## What to Avoid

- Generic AI-generated aesthetics
- Overused font families (Inter, Arial)
- Clichéd color schemes (generic blue gradients)
- Cookie-cutter design lacking context-specific character
- Excessive centered layouts, purple gradients, uniform rounded corners

## Implementation

Match code complexity to the vision:
- Maximalist → elaborate animations and effects
- Minimalist → precision in typography and spacing

Works across frameworks: React, HTML/CSS, Vue, etc. Target production-grade quality with memorable visual impact.
